Detailed Description

template<typename T, size_t BlockSize>
class allocators::BlockAllocator< T, BlockSize >

Allocator which allocates memory with blocks of constant size. It helps to reduce OS memory manager calls even with many allocation of short memory segments. Allocation of memory with size more than block size is prohibited.

Template Parameters
Ttype of the allocated elements
BlockSizesize of allocated block in terms of type T
